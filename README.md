# pepe

## 仓库说明

vue3 + ts + vite + electron 的模版项目

已包含的功能:

- request请求
- 请求mock
- ipcMain和ipcRenderer管理
- 多窗口打开管理
- electron-updater 全量更新应用(该功能需要后端服务配合)
- .npmrc解决electron下载失败
等

## 使用的技术

- electron-vite
- pnpm
- eslint
- prettier
- typescript
- vite
- commitlint
- husky
- vue3
- pinia
- axios
等
